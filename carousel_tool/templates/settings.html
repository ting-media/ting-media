📖 מדריך הגדרת Google Drive
  1. פתח Google Cloud Console וצור פרויקט חדש
  2. הפעל את Google Drive API
  3. צור Service Account → הורד מפתח JSON
  4. ב-Google Drive: צור תיקייה בשם carousels
  5. שתף את התיקייה עם כתובת האימייל של ה-Service Account
  6. העתק את ה-ID מה-URL של התיקייה למעלה
