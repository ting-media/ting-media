# 🚀 הוראות התחלה מהירות

## שלב 1: הכנת Gmail API

### צעדים:
1. עבור ל- [Google Cloud Console](https://console.cloud.google.com/)
2. צור פרויקט חדש
3. חיפוש ל-"Gmail API" והפעל אותו
4. לך ל-"Credentials" (כפתור ירוק בתפריט שמאל)
5. לחץ "Create Credentials" → "OAuth 2.0 Client ID"
6. בחר "Desktop application"
7. הורד את ה-JSON
8. שם את הקובץ בתיקיית הסוכן: `gmail_credentials.json`

## שלב 2: הכנת WhatsApp Business API

### צעדים:
1. עבור ל- [Meta for Developers](https://developers.facebook.com/)
2. צור חשבון עסק חדש (Business Account)
3. צור App חדש עבור WhatsApp
4. הוסף WhatsApp product
5. בחר ה-phone number שלך
6. שמור את:
   - **Access Token** (WHATSAPP_API_TOKEN)
   - **Phone Number ID** (WHATSAPP_PHONE_NUMBER_ID)
   - **Business Account ID** (WHATSAPP_BUSINESS_ACCOUNT_ID)

## שלב 3: הגדרת .env

1. פתח את הקובץ `.env.example`
2. ערוך והחלף את הערכים:

```bash
# 1. API Keys שלך
ANTHROPIC_API_KEY=sk-ant-... # שלך מ- Anthropic
WHATSAPP_API_TOKEN=EAA... # מ- Meta
WHATSAPP_PHONE_NUMBER_ID=1234567890123456 # מ- Meta
WHATSAPP_BUSINESS_ACCOUNT_ID=987654321098765 # מ- Meta

# 2. קבוצות WhatsApp לניתור
MONITORED_WHATSAPP_GROUPS=1234567890123456,9876543210987654

# 3. הגדרות מקומיות (אופציונאלי)
FLASK_PORT=5000
DEBUG=True  # כדי לראות errors בדאשבורד
```

3. שמור בשם `.env` (בלי .example)

## שלב 4: הרצה מקומית

### Windows:

```bash
run.bat

# או מ-terminal:
python main.py
```

### Mac/Linux:
```bash
bash run.sh
# או

```

## שלב 5: בדיקה

1. פתח דפדפן ל: **http://localhost:5000**
2. אתה אמור לראות דאשבורד עברי
3. לחץ על "הפעל סנכרון עכשיו" כדי לבדוק

## 🔧 בדיקת בעיות

### "בעיה עם Gmail"
```bash
# הסר את הקובץ הישן:
rm token.pickle
# ריצה שוב כדי להתחבר מחדש
python main.py
```

### "בעיה עם WhatsApp"
- בדוק ש-API Token תקף
- בדוק ש-Phone Number ID נכון
- וודא ש-Business Account פעיל

### "AttributeError" או שגיאות קוד
```bash
# התקן מחדש dependencies:
pip install -r requirements.txt
```

## 📊 הרצה ב-Heroku (ענן)

### צעדים:
1. התקן [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)
2. התחבר:
   ```bash
   heroku login
   ```
3. צור אפליקציה:
   ```bash
   heroku create <שם-אפליקציה>
   ```
4. הגדר environment variables:
   ```bash
   heroku config:set ANTHROPIC_API_KEY=sk-ant-...
   heroku config:set WHATSAPP_API_TOKEN=EAA...
   # וכו'...
   ```
5. Deploy:
   ```bash
   git push heroku main
   ```
6. צפה בדאשבורד:
   ```bash
   heroku open
   ```

## 📱 הגדרת Webhook (אופציונאלי)

אם אתה רוצה real-time messages (לא לחכות שעה):

1. ב-Meta for Developers, לך ל-WhatsApp → Configuration
2. הגדר Webhook URL:
   ```
   https://<הדומיין שלך>.herokuapp.com/api/webhook
   ```
3. Set Verify Token: `סדרת תווים חזקה`
4. בחר messages כ-event

## 📝 טיפים שימושיים

1. **הגדר keywords** - ב-`config.py` ערוך את `TASK_KEYWORDS` כדי להכיר משימות
2. **שנה אינטרוולים** - `SCHEDULER_INTERVAL_MINUTES` במשתנה environment כדי לשנות כמה פעמים לבדוק
3. **ייצוא נתונים** - `/api/export/summary` לקבל JSON של נתונים

## ❓ שאלות נפוצות

**Q: האם אני צריך WhatsApp Business Account?**
A: כן, זה חינם לפתיחה בעיסקי מטא. דרוש עבור API.

**Q: האם זה עולה כסף?**
A: Gmail API + Anthropic API עשויים להיות בעלויות קטנות בהשימוש כבד.

**Q: מה עם הפרטיות?**
A: כל הנתונים נשמרים ב-DB מקומי. מומלץ להשתמש בSSL בענן.

**Q: כיצד למחוק משימה?**
A: עדיין לא יש UI, שנה את הטבלה בDatabse ישירות (SQLite).
